Galleries

Two galleries are available that, when linked, automatically size the window to the image. These are built on the Fancybox framework; see Gallery 1 and Gallery 2.

Modal Window

An onclick function in boeing.js uses the button's id to target the corresponding content. The target for this example is located on this page just before the closing body tag. The modal window target in this example call an <iframe> to insert the .html page into the body of the parent page. Each modal window requires its own unique ID and can be accessed from the primary navigation, a text link or a button below.

Popovers

Add small overlays of content to any element for housing secondary information. Popovers whose both title and content are zero-length are never displayed. You can specify the direction (top, right, bottom, left) you want the popover to appear as well as if you want it to appear on click or on rollover.

Accordions

Accordion menus are used widely in navigation for minimizing and maximizing content. They assist the user to efficiently scan topics quickly and expand on the information they are looking for. Accordions are a practical, indespensible space saving technique that enables the user to access rich content in a small space.
