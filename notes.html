Boeing Responsive Web Template Notes

May 4, 2015

The Creative & Information Services Team is proud of the work that has been done to create a fully functional, next-generation, responsive web template.

When using this template please be aware that the Brand Zone requirements are different for the various media types. A smaller, graphical business unit identifier is used on the desktop and tablet media queries, while a text based identifier is used on the smart phone media query.

We track browser and device statistics bi-monthly. While mobile computing is growing by leaps and bounds, less than 1% of employees, contractors and suppliers are browsing the intranet using mobile devices at this time.
